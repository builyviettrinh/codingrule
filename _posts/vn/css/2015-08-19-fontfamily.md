Theo nguyên tắc, nếu không có chỉ định trong device font, thì chỉ định font như bên dưới:

```css
body {
  font-family: "メイリオ", Meiryo, "ヒラギノ角ゴ ProN W3", "Hiragino Kaku Gothic ProN", "游ゴシック", YuGothic, sans-serif;
}
```
