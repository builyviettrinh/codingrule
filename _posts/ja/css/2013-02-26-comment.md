大カテゴリ、小カテゴリを使い分け、適時区別してソースの視認性を確保してください。

ハイフンの数は40文字です。

```css
/*----------------------------------------
  主分類(ヘッダー、フッター、コンテンツなど)
----------------------------------------*/
.hentry {
  property: value;
}

/*  下層分類(適切な区別で)
----------------------------------------*/
#header {
  property: value;
}
```
