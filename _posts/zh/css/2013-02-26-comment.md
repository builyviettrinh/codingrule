请区别使用大标题、小标题，适时划分以确保源代码的可见性。

连字符的数量为 40 个字符。

```css
/*----------------------------------------
  Main classification (Header, footer contents, etc.)
----------------------------------------*/
.hentry {
  property: value;
}

/*  Lower classification (by distinguishing appropriately)
----------------------------------------*/
#header {
  property: value;
}
```
